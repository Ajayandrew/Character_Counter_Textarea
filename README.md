# Character_Counter_Textarea
An interactive textarea that updates a real-time character counter as the user types. The project features a maximum character limit, and warning messages when the limit is reached.

### Features
- Dark glass UI
- Real-time character counting
- Displays typed and remaining characters
- Maximum character limit enforcement
- Warning message when limit is reached
- Smooth responsive design

### Technologies Used
- HTML5
- Tailwind CSS
- JavaScript
